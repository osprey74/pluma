# Pluma Markdown プレビューのサンプル

Plumaの **Markdownプレビュー機能** のデモ用サンプルファイルです。

## 主な Markdown 記法

### 見出し

見出しは `#` の数でレベルが決まります（H1〜H6）。

### 文字装飾

- **太字（bold）**
- *斜体（italic）*
- `インラインコード`
- ~~取り消し線~~

### リスト

順不同リスト：

- 北海道
- 青森県
- 岩手県
  - 盛岡市
  - 花巻市

順序付きリスト：

1. ファイルを開く
2. 編集する
3. 保存する

タスクリスト：

- [x] ドキュメントを書く
- [x] スクリーンショットを用意する
- [ ] リリースノートを書く

### 表

| 機能 | ショートカット | 備考 |
|------|----------------|------|
| 新規作成 | Ctrl+N | 空のタブを開く |
| 開く | Ctrl+O | ファイル選択ダイアログ |
| 保存 | Ctrl+S | 上書き保存 |
| 印刷 | Ctrl+P | PDFとして出力 |

### コードブロック

```javascript
function greet(name) {
  console.log(`こんにちは、${name}さん！`);
}

greet("世界");
```

### 引用

> Plumaはコードエディタではなく、テキストエディタです。
> シンプルさを優先した設計になっています。

## プレビューの特徴

- **リアルタイム更新** — 編集と同時にプレビューが更新される
- **スクロール同期** — エディタとプレビューが連動してスクロール
- **リサイズ可能** — パネル境界をドラッグして幅を変更

Ctrl+Shift+M でプレビューの表示/非表示を切り替えられます。
